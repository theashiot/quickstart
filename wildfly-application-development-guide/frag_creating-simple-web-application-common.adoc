. Create a directory to store the Java files. The path in the command below is relative to __<application_home>__.
+
.Syntax
[source,options="nowrap",subs=+quotes]
----
$ mkdir -p src/main/java/<path_based_on_artifactID>
----
+
.Example with artifactID com.example.app
[source,options="nowrap",subs=+quotes]
----
$ mkdir -p src/main/java/com/example/app
----

. Navigate to the new directory.
+
.Syntax
[source,options="nowrap",subs=+quotes]
----
$ cd src/main/java/<path_based_on_artifactID>
----
+
.Example with artifactID com.example.app
[source,options="nowrap"]
----
$ cd src/main/java/com/example/app
----

. Create a file `HelloWorld.java` with the following content. If you used an `artifactID` different from `com.example.app`, change the package name to match the `artifactID`.
+
[source,java,options="nowrap"]
----
include::simple-webapp-example/src/main/java/com/example/app/HelloWorld.java[]
----

. Compile your application with the following command in __<application_home>__:
+
[source,options="nowrap"]
----
$ mvn package
----
