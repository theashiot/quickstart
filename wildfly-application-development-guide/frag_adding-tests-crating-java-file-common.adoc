. Create directories for tests from the project root directory. The directory containing the `pom.xml` file is the project root directory.
+
.Syntax
[source,options="nowrap",subs=+quotes]
----
$ mkdir -p src/test/java/__<path_based_on_artifactID>__
----
+
.Example
[source,options="nowrap"]
----
$ mkdir -p src/test/java/com/example/app
----

. Navigate to the new directory.
+
.Syntax
[source,options="nowrap",subs=+quotes]
----
$ cd src/test/java/<path_based_on_artifactID>
----
+
.Example
[source,options="nowrap"]
----
$ cd src/test/java/com/example/app
----
