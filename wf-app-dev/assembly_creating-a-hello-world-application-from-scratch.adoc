= Creating a hello world application from scratch

:context: creating-a-hello-world-application-from-scratch

Create a an application from scratch. We us Maven for dependency management. The application when compiled and deployed prints "Hello, world!".

Use the following procedures to create a simple "Hello World" application, deploy it on a running server and access the application in a browser:

* xref:creating-a-maven-project-for-a-simple-web-application_creating-a-hello-world-application-from-scratch[Create a Maven project for a simple web application]
* xref:creating-a-simple-web-application_creating-a-hello-world-application-from-scratch[Create a simple web application]

After you have deployed your application, you add tests to it. You can add a simple test using the following procedure:

* xref:adding-tests-to-web-applications_creating-a-hello-world-application-from-scratch[Adding tests to web applications]
